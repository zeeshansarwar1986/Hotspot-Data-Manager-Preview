## 🌍 Multilingual Description

### 🇬🇧 English
**Hotspot Data Manager** is an advanced network utility tool that provides real-time monitoring of all devices connected to your mobile hotspot. It features live speed tracking (Upload/Download) and detailed app-specific data usage reports, ensuring you have complete control over your network resources.

## 🚀 Key Features

- **#TrafficMonitor**: Real-time upload and download speeds for every connected device.
- **#DetailedReporting**: GlassWire-style app usage analysis (YouTube, WhatsApp, etc.).
- **#DeviceControl**: Instantly block or unblock connected clients.
- **#SmartAnalysis**: Visualize traffic history with dynamic charts.
- **#MaterialDesign**: Premium, state-of-the-art UI inspired by modern design principles.

---

## 🛠 Project Tools & Technologies

This application was built using a professional stack to ensure performance and scalability:

- **#Kotlin**: Core programming language for Android development.
- **#AndroidStudio**: Primary IDE used for building the application.
- **#MaterialComponents**: High-quality UI components (M3).
- **#ChartJS**: Used for the interactive web preview dashboards.
- **#RoomDatabase**: Local SQLite abstraction for performance-heavy data logging.
- **#TrafficStatsAPI**: Android system API for network traffic monitoring.
- **#HTML5 / #CSS3 / #JavaScript**: Powering the high-fidelity web-based preview system for developers.

## 💻 Tech Implementation Info

- **#NetworkStatsManager**: Implemented for precise per-UID data tracking.
- **#ViewModel**: Architecture pattern for state management.
- **#Coroutines**: Asynchronous polling of network statistics.
- **#DPI_Simulation**: Deep Packet Inspection simulation for usage categorization.
